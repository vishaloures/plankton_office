# 🦀 Подводный Офисный Tycoon (Voxel Edition)

MVP симулятора выживания в суровых реалиях подводного офиса. Игра выполнена в воксельном (Voxel) 3D-стиле и разрабатывается на движке **Godot Engine 4.3** с использованием модуля **godot_voxel**.

![Engine](https://img.shields.io/badge/Engine-Godot_4.6-blue.svg)

## 📋 Описание проекта

Вы управляете одним из обитателей подводного офиса (Крабом или Планктоном). Ваша цель — зарабатывать очки опыта, выполняя абсурдные задачи из Канбан-доски, и при этом не сойти с ума от выгорания, голода и офисной "душноты". 

### Основные фичи (CJM):
1. **Создание персонажа:**
   * **Форма:** Краб (с клешнями) или Планктон (циклоп).
   * **Профессия:** Программист (быстрый, но быстро выгорает), Менеджер (медленный, но экономный) или Дизайнер.
   * **Супер-навыки:** Пукун (газовые атаки), Сонный алкоголик, Токсичный сарказм.
   * **Генерация имени.**
2. **Менеджмент потребностей (The Sims-like):**
   * **Голод:** Удовлетворяется у Холодильника.
   * **Выгорание:** Снимается у Кулера с водой (или кофе).
   * **Душнота:** Лечится проветриванием Окна (Особенно важно для персонажей с навыком "Пукун").
3. **Рабочий процесс:**
   * Задачи выбираются из 3D Канбан-доски ("Починить кнопку", "Удалить легаси" и т.д.).
   * Интерактивность зависит от дистанции до объекта.
4. **Интерактивные Супер-навыки:**
   * При заполнении шкалы "Газы" (или другой), персонаж начинает биться в истерике. Клик по нему левой кнопкой мыши вызывает уникальную анимацию (например, взрыв воксельных частиц пука) и влияет на атмосферу в офисе.

## 🛠 Установка и Запуск (для Разработчиков)

Так как проект использует сложный C++ модуль для генерации вокселей, **стандартный редактор Godot не подойдет**. Вам нужна версия Godot со встроенным модулем `VoxelTerrain`.

### macOS
Мы подготовили скрипт для автоматической загрузки нужной версии редактора на Mac:

1. Откройте терминал в папке проекта.
2. Дайте скрипту права на выполнение и запустите его:
```bash
chmod +x setup_godot_mac.sh
./setup_godot_mac.sh
```
3. Скрипт скачает и распакует редактор в папку `GodotVoxel`.
4. Перейдите в Finder, откройте `GodotVoxel`, нажмите **Control+Клик (Правая кнопка)** на приложении `Godot` и выберите **"Открыть"** (чтобы обойти защиту macOS).
5. В открывшемся Godot нажмите **Import**, выберите файл `godot_plankton_tycoon/project.godot` и нажмите "Import & Edit".

### Windows / Linux
1. Скачайте нужную версию редактора со страницы релизов [Zylann/godot_voxel](https://github.com/Zylann/godot_voxel/releases). (Вам нужна версия `godot.windows.editor...` или `godot.linuxbsd.editor...`).
2. Распакуйте архив.
3. Запустите Godot и импортируйте проект из папки `godot_plankton_tycoon`.

## 🕹 Как играть (в редакторе)
1. Откройте проект в Godot.
2. Нажмите **Play** (Треугольник в правом верхнем углу) или клавиши `Cmd+B` (`Ctrl+B`).
3. В стартовом меню настройте своего персонажа и нажмите **"ПОДПИСАТЬ ОФФЕР"**.
4. Используйте **WASD** или **Стрелочки** для перемещения по офису.
5. Подойдите к рабочему столу, чтобы разблокировать кнопку "ОТКРЫТЬ КАНБАН".
6. Возьмите задачу и следите за шкалами! Если стало тяжело — бегите к кулеру, холодильнику или окну.
7. Если персонаж затрясся от "Супер-навыка" — быстрее кликайте по нему мышкой!

## 📂 Структура проекта
* `cjm.md` — Детальное описание пользовательского пути и механик.
* `design_system.md` — Описание визуального стиля (Voxel Rules, палитра).
* `setup_godot_mac.sh` — Установщик 3D-движка для macOS.
* `godot_plankton_tycoon/` — **Основной 3D проект на Godot**.
  * `global.gd` — Глобальное хранилище данных (Autoload).
  * `menu.tscn` / `menu.gd` — Сцена стартового меню и создания персонажа.
  * `main.tscn` / `main.gd` — Основная игровая 3D сцена (Офис) и UI менеджер.
  * `player.gd` — Скрипт контроллера, потребностей, движения и анимаций персонажа.
